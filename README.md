### Installation

Game requires [Node.js + npm](https://nodejs.org/) v4.5+ and [webpack](https://webpack.github.io/) v1.13+ to run.

```sh
$ git clone https://github.com/sns2009/puzzle.git
$ npm install
$ npm run production
$ npm run start
```
